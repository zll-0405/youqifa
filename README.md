## 为什么叫“有启发”

我很喜欢“有启发”这个词

我自己定义里面的“有启发”：这个点以前不知道/不了解/不清楚，或者被这个点打动

后来看到罗辑思维里面对它的定义是：

```
启发至少包含了四个互动过程：
第一，我自己得有一个问题。
第二，我在其他领域偶然遇到了一个新信息。
第三，这个新信息让我的那个老问题有了一个新答案。
第四，这个新答案，还能扩展成一个新思路，可以有更广泛的应用。
这四个心理同时发生，才能称之为是一个“启发”。

“启发”，指向自己的成长
```

觉得这个定义也很不错，所以决定开始做这个网站时，第一想到的就是它：“有启发”

希望这个网站的内容，能够给你一些启发
